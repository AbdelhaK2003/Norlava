## What is Norlava ?
 
Norlava is a real-world web application that combines fullstack development and artificial intelligence into one product. It is not a school project — it is a live platform designed, built, and deployed entirely from scratch.
 
The goal was simple : prove that modern web development and AI are not two separate worlds. Every part of this project — from the user interface to the backend logic to the AI layer — was built by one developer.
 
---
 
## Tech Stack
 
### Frontend
- HTML / CSS / JavaScript
- Responsive UI built from scratch
- Deployed and accessible at norlava.com
### Backend
- Node.js — REST API server (`dist-server/`)
- Handles routing, authentication, and data flow
- Scripts for automation and data processing (`scripts/`)
### AI Layer
- Integration with LLM APIs for intelligent features
- AI-powered processing connected to the fullstack architecture
- Designed so the AI layer plugs directly into the existing API — no separate service needed
### Infrastructure
- Deployed on a live server with a custom domain
- Version controlled with 246+ commits — continuous development
---
 
## Architecture
 
```
User (Browser)
      ↓
  Frontend (HTML/CSS/JS)
      ↓
  Node.js API (dist-server)
      ↓
  AI Layer (LLM integration)
      ↓
  Data / Scripts processing
```
 
The architecture follows the same principles used in production AI systems : a clean separation between the UI, the API, and the AI logic — making each layer independently maintainable and scalable.
 
---
 
## Why this project matters
 
Most developers either build web apps OR work with AI. Norlava was built to show that both can and should live together in the same product.
 
The same architecture principles used here — API layer, AI integration, secure data flow — are directly applicable to enterprise projects like RAG systems, intelligent document assistants, and AI-powered automation workflows.
 
This is the kind of project that comes from genuine curiosity, not a school assignment.
